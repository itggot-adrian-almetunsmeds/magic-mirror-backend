Setup

Google Calendar

  1. Navigate to developers.google.com

  2. Create a project

  3. Under the credentials tab click Create credentials and select OAuth Client ID

  4. Under application type select Other and if required fill in information

  5. Press the download button and save the file as credentials.json in the root directory

Running

§ rackup

or using rerun

§ rerun rackup

Errors

Error message: Unable to load the EventMachine C extension; To use the pure-ruby reactor, require 'em/pure_ruby' Solution: On Windows do gem uninstall eventmachine and then gem install eventmachine --platform=ruby
